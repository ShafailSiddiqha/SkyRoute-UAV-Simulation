SkyRoute is a real-time UAV route simulation system that visualizes how a drone navigates from a start point to a destination while avoiding obstacles.

## Features
- Real-time UAV movement
- Shortest, Safe, and Fast routing modes
- Dynamic obstacle addition and removal
- Live re-routing
- Scenario-based environments (Urban, Dense, Open)
- Metrics dashboard

## Tech Stack
- Frontend: React
- Backend: Python (Flask)
- APIs: REST
